# Customer Journey Map Builder

Interactive visualisation tool for mapping and analysing customer journeys with touchpoints, pain points, and opportunities.

## Features

- **Visual journey mapping** across multiple stages
- **Track touchpoints, emotions, and pain points**
- **Identify opportunities** for improvement
- **Customisable journey stages** and phases
- **Real-time editing** and updates
- **Channel analytics** showing distribution of touchpoints

## Live Demo

Visit [https://customer-journey-map-two.vercel.app](https://customer-journey-map-two.vercel.app)

## Tech Stack

- React 18
- Vite
- Tailwind CSS

## Getting Started

```bash
npm install
npm run dev
```

## How It Works

1. **Set up your journey** by defining the journey name
2. **Add stages** (e.g., Awareness, Consideration, Purchase, Retention)
3. **Add touchpoints** to each stage with:
   - Touchpoint name
   - Channel (Email, Website, Mobile App, Social Media, etc.)
   - Customer emotion (Happy, Neutral, Frustrated, Confused, Delighted)
   - Optional description
4. **View analytics** including:
   - Total touchpoints
   - Channels used
   - Channel distribution

## Use Cases

- **CRM Planning**: Map customer lifecycle communications
- **UX Design**: Identify pain points in user experience
- **Marketing Strategy**: Optimise multi-channel campaigns
- **Customer Success**: Improve onboarding and retention
